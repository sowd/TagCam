## Tag Camera (タグカメラ)

タグカメラは、写真に様々な補足情報を追加するための、Android用カメラアプリです。商用ではなく研究目的のソフトウェアです。

Mark Harman氏による[Open Camera](https://opencamera.sourceforge.io/)のソースコードをベースに開発しており、元のライセンスがGPL3なので、こちらも[GPL3](LICENSE)です。
